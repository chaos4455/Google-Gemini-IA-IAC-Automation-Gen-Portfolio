## 🧠 Inteligência Artificial: O Poder da Transformação 🧠

### 🤔 O Que é IA? Uma Nova Definição
A Inteligência Artificial é a capacidade de um sistema simular o raciocínio humano, aprendendo, tomando decisões e resolvendo problemas, criando novas possibilidades e oportunidades para o mundo dos negócios.

### 🗣️ LLMs: A Nova Linguagem das Máquinas
Os Modelos de Linguagem Grandes (LLMs), como o Google Gemini, são a base dos chatbots avançados, assistentes virtuais e sistemas de processamento de linguagem natural. Eles entendem e geram texto de maneira semelhante a humanos, revolucionando a interação com as máquinas.

### ✨ Minhas Técnicas Exclusivas: O Segredo do Sucesso
Minhas soluções de IA são construídas com:
- **Fine-tuning:** Adaptação de modelos pré-treinados para tarefas específicas, otimizando o desempenho com precisão.
- **Hyperparameter Embedding:** Otimização de parâmetros cruciais para resultados superiores e consistentes.
- **Técnicas Próprias:** Abordagens inovadoras que maximizam o potencial da IA, elevando a qualidade das soluções.

### 💡 Aplicações Práticas da IA: Transformando seu Negócio
A IA pode ser aplicada em diversas áreas da sua empresa, como:
- **🤖 Atendimento ao Cliente:** Chatbots inteligentes que oferecem suporte 24/7.
- **⚙️ Automação de Processos:** Sistemas que automatizam tarefas repetitivas, liberando sua equipe para atividades estratégicas.
- **📊 Análise de Dados:** Ferramentas que extraem insights valiosos para tomada de decisões assertivas.
 - **📝 Geração de Templates e Configurações:** Ferramentas de IA que criam automaticamente arquivos de configuração para infraestrutura como código (IaC), acelerando a implantação e reduzindo erros.

## 💻 DevOps e IaC: A Base para a Inovação 💻

### 🤝 DevOps: Agilidade e Colaboração em Harmonia
DevOps é uma cultura que integra desenvolvimento e operações, promovendo a colaboração e a entrega contínua de software, trazendo agilidade e eficiência para a sua empresa.

### 🧱 IaC: Infraestrutura Como Código, a Base do Futuro
Infraestrutura como Código (IaC) é a prática de gerenciar e provisionar infraestrutura através de código, permitindo tratar sua infraestrutura da mesma forma que trata seu software, garantindo maior consistência, escalabilidade e segurança.

Um projeto de portfolio que combina processamento de PDFs com técnicas avançadas de visão computacional, NLP e processamento paralelo para extrair imagens e aplicar marcações semânticas. Este é o  meu  portfólio de projetos , que integra diversas técnicas de ponta para extrair o máximo de valor de documentos PDF.

## 🚀 Funcionalidades Principais

- **🖼️ Extração de Imagens:** Converta páginas de PDFs em imagens de alta qualidade com facilidade.
- **🔥 Mapas de Calor:** Aplique mapas de calor semânticos para identificar áreas importantes em imagens.
- **🤖 Reconhecimento Óptico de Caracteres (OCR):** Extraia texto de imagens usando EasyOCR.
- **🧠 Análise Semântica com BERT:** Identifique blocos de texto similares usando embeddings BERT.
- **🧵 Processamento Paralelo:** Maximize a eficiência usando threads para processar múltiplas páginas e arquivos simultaneamente.
- **✨ Streamlit Interface:** Interface web interativa para uso fácil e rápido.
- **✅ Download de ZIP:** Baixe todas as imagens processadas em um único arquivo ZIP.
- **🎨 Paletas de Cores:** Visualize com diferentes paletas de cores e efeitos de sobreposição, como *viridis*, *inferno* e *magma*.
- **🎛️ Configurações Flexíveis:** Ajuste parâmetros como o raio de *blur*, limiar de *threshold* e transparência dos *heatmaps*.
- **✨ Apresentação em Tempo Real:** Imagens são exibidas em tela enquanto o processamento acontece.
- **⚙️ Uso Otimizado de Recursos:** Monitoramento em tempo real de uso de CPU e memória.

## 🛠️ Tecnologias Utilizadas
  
  *  **Python:** Linguagem de programação principal
  *  **Streamlit:** Criação da interface web.
  *  **PyMuPDF (fitz):** Processamento de arquivos PDF.
  *  **Pillow:** Manipulação de imagens.
  *  **NumPy:** Operações numéricas.
  *  **SciPy:** Implementação de filtro gaussiano.
  *  **Scikit-image:** Processamento e análise de imagens.
  *  **EasyOCR:** Extração de texto de imagens.
  *  **Transformers (BERT):** Modelo para embeddings de texto.
  *  **Torch:** Biblioteca para operações de tensores.
  *  **Scikit-learn:** Ferramentas para agrupamento de dados (clustering)
  *  **Psutil:** Monitoramento de recursos do sistema.
  *  **Matplotlib:** Criação de paletas de cores para visualização de mapas de calor.
  *  **Rich:** Criação de Logs e paineis ricos no terminal(Utilizada no Script original, porém, neste, não é necessária no contexto do Streamlit)
  *  **Pandas:** Estruturação e organização de dados.
